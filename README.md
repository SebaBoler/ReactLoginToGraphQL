# ReactLoginToGraphQL

Sample application using Apollo-client with react and GraphQL-Yoga 

# Installation

## Backend
```
1. cd backend
2. npm install or yarn install
3. cd prisma
4. docker-compose up -d ( need docker )
5. cd ..
6. prisma deploy
7. prisma seed
8. yarn dev
```
Congratulation your server GraphQL working

## Frontend
```
1. cd frontend
2. npm install or yarn install
3. yarn start
``
